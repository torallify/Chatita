## üé¨ Recording Tips

### Before Recording:
1. **Clean your data** - Start fresh or use demo data
2. **Test the flow** - Practice 2-3 times
3. **Prepare food photos** - High quality, well-lit meals
4. **Set demo user name** - Something relatable (e.g., "Sarah", "Maria")
5. **Pre-log some meals** - So Insights screen has data

### During Recording:
1. **Record in landscape** for desktop viewing
2. **Move slowly** - Give viewers time to see each screen
3. **Hold on important moments** - Let numbers/text settle
4. **Avoid quick taps** - Make interactions visible
5. **Record multiple takes** - Choose the best one

### What to Record:
**Priority Shots (Must Have):**
- Home screen with greeting
- Full meal analysis flow (camera ‚Üí AI ‚Üí results)
- Insights screen with stats
- Rewards screen with milestones

**Nice to Have:**
- Language toggle in action
